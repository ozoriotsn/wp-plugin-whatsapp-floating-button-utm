Easily add a floating WhatsApp button to your WordPress site to let visitors contact you instantly.

== Description ==
This plugin Add a WhatsApp floating button on specific pages with customizable phone number, UTM, and page targets.

**Features:**
- Add your WhatsApp number easily.
- Choose button position (left or right).
- Mobile friendly.
- Lightweight and fast.
- Insert UTM params
- Not show button in specific pages 

== Installation ==

1. Upload the plugin folder to the `/wp-content/plugins/` directory, or install it directly through the WordPress plugins screen.
2. Activate the plugin through the 'Plugins' screen in WordPress.
3. Go to **Settings > WhatsApp Floating Button UTM ** to configure your phone number and button options.

== Frequently Asked Questions ==

= Can I customize the button position? =
Yes, you can choose left or right in the settings.

= Does it work on mobile? =
Yes, it is fully responsive and works perfectly on mobile devices.
